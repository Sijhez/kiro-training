# Product Overview

## Micro Blogging App

A serverless social media platform built on AWS that enables users to share short posts, follow other users, and engage with content through likes and interactions.

## Core Features

- User authentication and profile management
- Post creation and feed viewing
- Social interactions (likes, follows/unfollows)
- Real-time follower counts and engagement metrics
- Responsive design for mobile and desktop

## Architecture

Serverless application using AWS services:
- Frontend: React SPA hosted on S3/CloudFront
- Backend: Lambda functions with API Gateway
- Auth: AWS Cognito for user management
- Database: DynamoDB for data persistence
- Infrastructure: AWS CDK for IaC

## User Experience

Clean, modern social media interface with purple accent colors, rounded elements, and a focus on content discovery and user engagement.
